Interview process

  1. Multilevel Sensor Get Supported Sensor

  2. For each supported Sensor Type: Multilevel Sensor Get Supported Scale

  3. For first Scale Supported: Multilevel Sensor Get Command

Note

The structure of the attribute store is :

|__ SENSOR_TYPE
|   |__ SUPPORTED_SCALES
|   |__ SCALE
|   |__ SENSOR_VALUE
|__ SUPPORTED_SENSOR_TYPES
