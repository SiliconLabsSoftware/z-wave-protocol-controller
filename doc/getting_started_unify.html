Introduction

Z-Wave-Protocol-Controller is a software framework that simplifies the developer experience, removing difficult parts of network control and network management as it relates to gateway and hub development in IoT applications.

Prerequisites

  • An RPi4 preinstalled with Raspberry Pi OS Linux Version 12, Bookworm 64-bit

  • At least one of the following, supported wireless radios is connected to the RPi4 via USB:

    • Z-Wave module flashed with Z-Wave - NCP Serial API Controller.

  1. Use Unify on Raspberry Pi for complete hands-on of Unify IoT gateway.
